# Web Lite Plugin Deprecation Notice

## Important Update

**The i18n Translate Lite: Web plugin is deprecated as of November 2025.**

No further updates or bug fixes will be provided for the Lite version. We strongly recommend migrating to the new **i18n Translate Pro: Web** plugin, which now includes a **FREE tier** with all the features previously available in Lite.

## Why This Change?

We've transitioned to a **freemium model** that provides:

- **FREE tier**: All Lite features (Smart Text to i18n + single language translation) available at no cost
- **Pro tier**: Unlimited languages and advanced workflow features
- **Better support**: All users on one platform receive updates and improvements
- **Simplified ecosystem**: One plugin instead of two separate versions

## What This Means for You

### Current Lite Users

You can continue using the Lite plugin as-is, but:
- ❌ No new features will be added
- ❌ No bug fixes will be provided
- ❌ No compatibility updates for future IntelliJ/WebStorm versions
- ✅ The plugin will continue to work on currently supported IDE versions

### Recommended Action

**Switch to i18n Translate Pro: Web** to enjoy:
- ✅ All your current Lite features remain FREE forever
- ✅ Regular updates and bug fixes
- ✅ Future IDE version compatibility (IntelliJ Ultimate, WebStorm)
- ✅ Option to upgrade to Pro tier for unlimited languages

## Migration Steps

### 1. Install Pro Plugin

Visit the [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/28020-i18n-translate-pro-web) and install **i18n Translate Pro: Web**.

### 2. Uninstall Lite Plugin

Go to **Settings → Plugins**, find **i18n Translate Lite: Web**, and uninstall it.

### 3. Reconfigure Settings

**Important**: Settings and API keys will NOT be preserved during migration.

After installing the Pro plugin:
- Re-enter your OpenAI/Anthropic API key in **Settings → Tools → i18n Translate Pro: Web**
- Reconfigure any custom settings (model selection, temperature, etc.)

### 4. Restart IDE

Restart your IDE (IntelliJ Ultimate or WebStorm) to complete the migration.

### 5. Enjoy FREE Features

The Pro plugin automatically provides FREE access to:
- Smart Text to i18n extraction (fully functional)
- Full LLM translation with your API key
- Single target language translation
- Support for Next.js, Vue i18n, Angular i18n, and generic JSON formats

To unlock unlimited languages and advanced features, you can upgrade to Pro tier anytime.

## Alternative Access Methods

While the commit panel translation feature is not available in the FREE tier, you can still access all core translation features through:

- **Right-click files** in Project View → "Translate JSON i18n File" or "Translate Angular XLF i18n File"
- **Tools menu** → "i18n Translate Tools"
- All core translation features work normally
