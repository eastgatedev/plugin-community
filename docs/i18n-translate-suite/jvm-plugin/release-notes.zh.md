# 发布说明 - i18n 翻译专业版：JVM

> 🌐 **Language / 语言**: [🇺🇸 English](release-notes.md) | [🇨🇳 简体中文](release-notes.zh.md)

## 版本 2025.1.8 <small>(2025-11-17)</small>

### 🤖 LLM 模型更新
- **新增 21 个模型：** GPT-5 系列（Nano/Mini/Pro）、GPT-4.1 系列、Claude 4.5（Sonnet/Haiku）、Claude Opus 4.1、DeepSeek R1、Phi4 等，覆盖所有 8 个提供商
- **120+ 总模型数：** 全面的模型覆盖，包括 Google Gemini 2.5、OpenAI GPT-5、Anthropic Claude 4.5、DeepSeek、Azure OpenAI、OpenRouter、Ollama 和自定义提供商
- **智能成本优化：** 新模型以更低的成本提供更好的性能 - GPT-5 Nano 提供最佳性价比

### 💰 旧模型检测
- **自动过时模型警报：** 插件现在会检测您是否使用旧模型，并提供智能升级建议
- **成本节省智能：** 显示价格对比，帮助节省高达 99.6% 的翻译成本（例如：GPT-4 → GPT-5 Nano：每百万 token 从 $40.00 降至 $0.17）
- **价格对比仪表板：** 查看当前模型成本与推荐替代方案的详细节省分析
- **基于版本的忽略：** 对话框在下次插件更新并提供新模型建议之前不会再次显示

### 🔍 OpenRouter 模型浏览器
- **300+ 模型触手可及：** 新对话框可浏览和比较所有可用的 OpenRouter 模型
- **实时搜索和筛选：** 按名称、提供商或功能查找模型，即时显示结果
- **价格智能：** 比较不同模型的输入/输出 token 成本
- **上下文窗口信息：** 查看每个模型的最大上下文窗口大小
- **智能推荐：** 按受欢迎程度和性价比对模型进行排序

### ⚡ API 密钥设置改进
- **重新设计的提供商选择：** 清晰对比免费（Gemini）和高级选项
- **单击设置：** 简化的工作流程，带有"设置"按钮，默认为推荐的 Gemini 提供商
- **可视化提供商对比：** 并排对比框显示功能和优势
- **列出所有提供商：** 提前显示所有 8 个支持的 LLM 提供商的完整列表

### 🐛 错误修复
- **Spring 依赖面板：** 修复了提取对话框中 Spring 依赖面板不显示的问题
- **LLM 翻译可靠性：** 改进了错误处理和重试逻辑，以获得更稳定的翻译操作
- **配置持久性：** 更好地处理 IDE 重启后的 API 密钥和提供商设置

**专业定价智能：** 通过自动旧模型检测保持领先！插件现在会在您使用过时模型时发出警报，并推荐可节省高达 99.6% 成本同时提供更好性能的现代替代方案！

## 版本 2025.1.4 <small>(2025-08-17)</small>

- **🎯 智能文本转 i18n：** 革命性新功能，通过键盘快捷键 Alt+Shift+] 提取硬编码字符串并转换为 i18n 方法调用
- **⚡ 即时代码转换：** 自动检测字符串字面量，生成 Spring MessageSource 集成并提供预览对话框
- **🔧 智能代码增强：** 当方法缺少 messageSource 和 locale 参数时自动注入
- **📝 开发者生产力：** 即时消除硬编码字符串 - 选择文本或定位光标并按 Alt+Shift+] 立即进行 i18n 转换
- **🚀 IntelliJ IDEA 2025.1+ 支持：** 最低所需平台版本升级到 2025.1
- **⚡ K2 Kotlin 编译器模式：** 全面支持 Kotlin 的新 K2 编译器，具有增强的性能和稳定性
- **🔍 改进的检查：** 从 K2 模式访问 100+ 增强的代码检查和意图
- **🛠️ 平台现代化：** 利用最新的 IntelliJ 平台 API 以获得更好的兼容性

**智能文本转 i18n 革命：** 改变您的 i18n 工作流程！不再需要手动字符串提取 - 只需按 Alt+Shift+]，看着硬编码文本变成带有自动 MessageSource 集成的正确 i18n 方法调用！
