# JVM Lite Plugin Deprecation Notice

## Important Update

**The i18n Translate Lite: JVM plugin is deprecated as of November 2025.**

No further updates or bug fixes will be provided for the Lite version. We strongly recommend migrating to the new **i18n Translate Pro: JVM** plugin, which now includes a **FREE tier** with all the features previously available in Lite.

## Why This Change?

We've transitioned to a **freemium model** that provides:

- **FREE tier**: All Lite features (Smart Text to i18n + single language translation) available at no cost
- **Pro tier**: Unlimited languages and advanced workflow features
- **Better support**: All users on one platform receive updates and improvements
- **Simplified ecosystem**: One plugin instead of two separate versions
